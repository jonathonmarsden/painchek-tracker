  2. DEPLOY TO NETLIFY (Choose One Method)
     ─────────────────────────────────────

     A) Drag & Drop (Easiest - 60 seconds)
        • Go to: https://app.netlify.com/drop
        • Drag the painchek-tracker folder
        • Change site name to "painchek-tracker"
        • Done! Access at: https://painchek-tracker.netlify.app

     B) Command Line (If you have npm)
        • Open Terminal
        • cd /Users/jonathonmarsden/Projects/painchek-tracker
        • npm install -g netlify-cli
        • netlify login
        • netlify deploy --prod


  3. ADD TO iOS HOME SCREEN
     ───────────────────────
     • Open Safari on iPhone/iPad
     • Go to https://painchek-tracker.netlify.app
     • Tap Share button
     • Tap "Add to Home Screen"
     • Name it "PainChek"
     • Tap Add
